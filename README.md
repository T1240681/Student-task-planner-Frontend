Student Task Planner Frontend

Click here to watch the YouTube demo and see how the interface looks: https://youtu.be/_WBZuOtQFy4

A responsive student task planner built using React and TypeScript. This project helps users manage coursework and personal tasks with task creation, editing, searching, filtering, priority selection, completed task tracking, and browser storage using localStorage.

FEATURES

Create new coursework or personal tasks
Edit existing tasks
Delete tasks
Search tasks by title or module name
Filter tasks by completion status
Filter tasks by priority level
Mark tasks as completed or not completed
Save tasks in the browser using localStorage
Responsive user interface for different screen sizes
Smooth animations for task cards and sections

TECHNOLOGIES

React
TypeScript
HTML
CSS
localStorage
Vite

HOW TO RUN

Clone the repository:
git clone https://github.com/T1240681/Student-task-planner-Front-end-.git
Open the project folder:
cd Student-task-planner-Front-end-
Install dependencies:
npm install
Start the development server:
npm run dev
Open the local host link shown in the terminal.
